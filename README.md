# Provided / Built-in Features

1) BMI Calculator
2) BMR Calculator
3) Unit Converters (length, weight, speed and many more)
4) Calculator
5) Compass
6) Cryptography (Hash Generator)
7) EMI and Interest rate Calculator
8) Password Generator
9) QR code scanner
10) QR code generator
11) TODO List
12) Stopwatch
